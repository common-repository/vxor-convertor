F2blog.cont for vXor Convertor

== 注意事项 ==

* 请在使用前仔细阅读 vXor Convertor 的安装、使用及 FAQ。

* 请根据您的原博客数据库的具体情况如实设置。
  每次转换记录条数建议为 100，不宜过大。

* 附件文件夹。
  附件位于 wp-content/uploads/xxx。xxx 为您所设置的附件文件夹。
  请手动将原博客的附件文件夹 attachments 里面的全部文件复制到 wp-content/uploads/xxx 中。

* 加密日志密码。
  原博客加密日志密码无法用于 WordPress，需重新设置。
  原博客原有加密日志密码将重置为您所设置项。

* 原博客时区。
  用于转换日志发布时间，请如实填写。

* 页面。
  添加“关于”页面为 WordPress 自带默认页面。
  “留言簿”为原博客留言所使用的页面，若不添加，则不转换留言。

* 博客网址。
  原始值为原始的网址，替换值为新的网址。用于本地转换时使用。
  若不填写，则以当前 WordPress 网址做为替换值。

* 日志替换内容。
  用于替换日志中的内容。
  如将 ABC 替换成 CBA，则在原始值中填写 ABC，在替换值中填写 CBA。
  可添加无限个日志替换内容。

* 转换错误提示。
  请将错误提示信息反馈给插件作者。

* 仅管理员可见的评论、留言，转换后默认为未通过审核的评论及留言。请根据个人情况进行审核。

* 若日志中有仅注册会员可下载的附件，则该日志设置为“私有的”。
